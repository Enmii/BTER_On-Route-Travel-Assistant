README

On-Route Travel Assistant is an Android based mobile application used for route finding specifically for commuting routes within the 4th District of Metro Manila. This mobile application was developed with the Google Maps API and uses the A* search algorithm to find routes for public transport. For more information regarding the usage of the application can be read on the user manual. 


REQUIREMENTS

Operating System: Android OS: Jelly Bean (Android 4.3) or higher is supported

Processor: 1.0 GHz single-core CPU

Disk: 300MB free space

RAM: 512MB or higher

Graphics: Direct-X 9.0 or higher

Network: GSM, 3G, HSPA, HSPA+, LTE with GPS functionality.
