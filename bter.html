On Route Travel Assistant using A* Search Algorithm

Disclaimer

This project was developed by Big Tech Energy Reformed. The project's goal is to develop a routing application that allows its users to find public transport routes within the 4th District of Metro Manila. This application uses Google's geolocation services to find the user's current location within the map. Please keep in mind that the application only takes the device's location as private data and the application does not keep user data in its system.

Minimum Requirements

Operating System: Android OS: Jelly Bean (Android 4.3) or higher is supported

Processor: 1.0 GHz single-core CPU

Disk: 300MB free space

RAM: 512MB or higher

Graphics: Direct-X 9.0 or higher

Network: GSM, 3G, HSPA, HSPA+, LTE with GPS functionality.
